# Karan Kumar - Professional Portfolio Website

A modern, high-performance, glassmorphic portfolio website designed for Karan Kumar.

## Features
- **Modern Glassmorphic UI**: Translucent cards, blue-black gradient glow animations, and Outfit & Inter typography.
- **Dynamic Interactions**: Smooth scroll behaviors, sticky navbar scroll transitions, animated typing text in hero section.
- **Fully Responsive**: Media queries targeting desktop, tablet, and mobile screens.
- **SEO Optimized**: Pre-configured meta tags, semantic markup, and descriptive titles.
- **Contact Form**: Implemented frontend form validation and simulated asynchronous submitting behavior.

## Project Structure
- `index.html`: Contains SEO configuration, section semantics, and component references.
- `style.css`: Contains CSS variables, layout positioning, glassmorphic utilities, and custom animation keyframes.
- `script.js`: Controls active navigation elements, typing animations, and contact form handling.
- `Karan_Kumar_Resume.pdf` (Placeholder): Replace this file with your actual PDF resume.

## How to Run Locally
Simply open `index.html` in any browser, or use a local dev server (e.g., Live Server in VS Code, or python server):
```bash
python -m http.server 8000
```
Then navigate to `http://localhost:8000`.

## Deployment
This website is fully deploy-ready. You can deploy it to **Vercel**, **GitHub Pages**, or **Netlify** by uploading these repository files directly.
